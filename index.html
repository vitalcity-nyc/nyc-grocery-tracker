PUBLIC SUPERMARKET ACCOUNTABILITY

Where are New York City's supermarkets?

Licensed retail food stores in the five boroughs, sized by reported square footage. Mayor Mamdani has said the city will open one public supermarket in each borough; the first is planned for La Marqueta in East Harlem. This map is a baseline for asking which neighborhoods have a real supermarket today and which do not. The underlying state license file does not distinguish supermarkets from pharmacies, dollar stores, breweries, or bakeries; this map applies a name-based filter, but it is imperfect. Data current as of September 2025; new openings (e.g., Lidl at 120 5th Ave, Brooklyn) are missing. Methodology & sources →
